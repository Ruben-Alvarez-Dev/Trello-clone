<ï¿½ How to Use

  1. Add Lists: Click "Add a list" to create new columns
  2. Add Tasks: Click "Add a task" within any list to create new tasks
  3. Drag & Drop:
    • Drag tasks between lists to change their status
    • Reorder tasks within the same list
    • Reorder lists horizontally
  4. Edit: Double-click on any task or list title to edit
  5. Delete: Click the trash icon to remove tasks or lists
