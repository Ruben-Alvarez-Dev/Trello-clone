Custom hook for persistent localStorage state management

Provides stateful localStorage management with automatic synchronization and error handling. Returns a stateful value and setter function similar to React's useState, but with localStorage persistence.

Examples
// Basic usage
const [name, setName] = useLocalStorage('userName', 'Anonymous');
// With objects/arrays
const [todos, setTodos] = useLocalStorage('todos', []);
// Error handling is automatic
const [data, setData] = useLocalStorage('data', null);
